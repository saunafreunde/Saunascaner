# Hintergrundbilder

Lege hier deine Schwarzwald-Fotos ab:

- `forest-1.jpg` → wird auf `/dashboard` als Hintergrund verwendet
- `forest-2.jpg` → wird auf `/`, `/planner` als ruhigerer Hintergrund verwendet

**Empfohlene Größe:** 1920×1080 oder größer, JPG, ~200–500 KB optimiert.

**Quellen für lizenzfreie Schwarzwald-Bilder:**
- https://unsplash.com/s/photos/black-forest-germany
- https://www.pexels.com/search/black%20forest/
- https://pixabay.com/images/search/schwarzwald/

Suchbegriffe: *black forest*, *schwarzwald*, *tannen nebel*, *forest fog germany*, *nordschwarzwald*

Wenn die Dateien fehlen, zeigt die App einen tiefgrünen Fallback-Verlauf — funktioniert ohne Bilder.
